# LogiTruck - Logistics & Transportation Website

A professional, responsive website for LogiTruck logistics and transportation services.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations and transitions
- **Service Showcase**: Comprehensive display of logistics services offered
- **Contact Form**: Interactive quote request form for customer inquiries
- **Smooth Navigation**: Smooth scrolling and intuitive navigation menu
- **Mobile-Friendly**: Hamburger menu for mobile devices

## Services Offered

- Freight Shipping
- Full Truckload (FTL)
- Warehousing & Storage
- International Shipping
- Express Delivery
- Real-Time Tracking

## How to Use

Simply open `index.html` in any modern web browser to view the website.

### Viewing Locally

1. Clone the repository
2. Open `index.html` in your web browser
3. No build process or dependencies required!

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript
- Responsive Design principles
